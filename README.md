# Планировщик задач

## Описание приложение

Это выпускной проект курса "Go-разработчик с нуля" Яндекс Практикума. Это аналог TODO-листа с довольно простым функционалом:

- добавление, редактирование и удаление задач
- просмотр общего списка задач
- авторизация при наличии заданного пароля приложения

## Задания со звёздочкой

Для данного проекта были выполнены все задания со звёздочкой

## Инструкции по запуску

Перед тем как запустить приложение, стоит обратить внимание на `.env`, в котором нужно задать переменные окружения со своими значениями. Если не указана переменная окружения `TODO_PORT`, то приложение будет доступно по адресу `http://localhost:7540`.

```bash
# Основные команды управления приложением

# Сборка и запуск
docker compose up --build

# Останов
docker compose down
```

## Тестирование приложения

В файле `tests/settings.py` перед запуском тестов следует указывать следующие параметры:

- `Port` - усли хочется использовать другой порт, на котором будет работать приложение
- `DBFile` - расположение файла БД sqlite
- `FullNextDate` - для тестов расширенного вычисления следующей даты
- `Search` - для тестов расширенного поиска
- `Token` - для тестов аутентификации и авторизации сюда нужно вставить значение токена, например, из cookie

Команда для запуска тестов `go test ./tests`
